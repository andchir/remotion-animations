# Описание текстовой анимации

## Общее описание

Анимация представляет собой плавное появление и исчезновение блока с именем и должностью персонажа, расположенного в левом нижнем углу кадра.

## Визуальные элементы

### Расположение
- **Позиция**: левый нижний угол кадра
- **Отступы**: примерно 40-50px от левого края и 40-50px от нижнего края

### Структура блока
Блок состоит из двух элементов:
1. **Имя персонажа** (верхняя строка): "ISLA KING"
   - Цвет текста: белый (#FFFFFF)
   - Шрифт: жирный, sans-serif
   - Размер: крупный (примерно 24-28px)
   - Фон: черный прямоугольник с белым текстом

2. **Должность** (нижняя строка): "ENGINEER"
   - Цвет текста: белый (#FFFFFF)
   - Шрифт: жирный, sans-serif
   - Размер: меньше имени (примерно 14-16px)
   - Фон: оранжевый прямоугольник (#FF8C00 или подобный)

### Визуальный стиль
- Оба текстовых элемента имеют прямоугольный фон
- Текст имеет небольшие внутренние отступы (padding) от краев фона
- Между именем и должностью практически нет вертикального отступа (элементы плотно прилегают друг к другу)
- Общая форма: компактный двухстрочный блок

## Детальное описание анимации

### Фаза 1: Появление (кадры 1-30)

**Кадры 1-10**: Сцена без текста
- Фоновое видео показывает городской пейзаж (небоскребы)
- Текстовая анимация еще не началась

**Кадры 11-20**: Появление прогресс-бара
- В левом нижнем углу начинает появляться тонкая белая горизонтальная полоса (прогресс-бар)
- Под полосой появляется оранжевая линия-индикатор, которая начинает расти слева направо
- Анимация очень плавная, с использованием easing функции (скорее всего ease-out или ease-in-out)
- Белая полоса служит фоном для будущего имени
- Оранжевая полоса-индикатор заполняется примерно на 30-40% ширины белой полосы

**Кадры 20-30**: Формирование текста имени
- Белый прямоугольник расширяется до полной ширины
- Начинает появляться текст "ISLA KING" (побуквенная анимация или плавное проявление)
- Текст появляется с эффектом fade-in (плавное увеличение непрозрачности)
- Одновременно оранжевая полоса продолжает расширяться под именем
- На кадре 20 виден частичный текст "ISLA KIN" с оранжевым индикатором "ENGIN" под ним
- На кадре 25 виден полный текст "ISLA KIN" с оранжевым блоком "ENGINEER" под ним

**Кадры 30-40**: Полное отображение
- На кадре 30 текст "ISLA KING" отображается полностью на черном фоне
- Оранжевый блок с текстом "ENGINEER" полностью сформирован и отображается под именем
- Анимация появления завершена
- Блок находится в статическом состоянии (hold state)

### Фаза 2: Статическое отображение (кадры 31-50)

**Кадры 31-50**: Удержание
- Текстовый блок остается на экране без изменений
- Имя "ISLA KING" на черном фоне
- Должность "ENGINEER" на оранжевом фоне
- Фоновое видео продолжает воспроизводиться
- Длительность статического отображения составляет примерно 20 кадров (около 0.67 секунды при 30fps)

### Фаза 3: Исчезновение (кадры 51-60)

**Кадры 51-55**: Начало исчезновения
- Текстовый блок начинает плавно исчезать
- Используется эффект fade-out (уменьшение непрозрачности)
- Сначала исчезает должность "ENGINEER" (оранжевый блок)
- Текст становится менее четким, фон становится более прозрачным

**Кадры 56-60**: Полное исчезновение
- Продолжается fade-out эффект
- Имя "ISLA KING" также начинает исчезать
- К кадру 60 текстовый блок практически полностью невидим
- Остается только фоновое видео без текстовых элементов

## Технические параметры для воссоздания

### Таймлайн (при 30 fps)
- **0-10 кадров (0-0.33с)**: Пустой экран, подготовка
- **11-20 кадров (0.33-0.67с)**: Анимация появления прогресс-бара и начало появления текста
- **21-30 кадров (0.67-1.0с)**: Завершение появления текста
- **31-50 кадров (1.0-1.67с)**: Статическое отображение
- **51-60 кадров (1.67-2.0с)**: Плавное исчезновение

### Параметры анимации

**Появление (fade-in + slide-in)**:
- Длительность: ~0.67 секунды (20 кадров)
- Easing: ease-out или cubic-bezier(0.25, 0.1, 0.25, 1)
- Эффекты:
  - Opacity: 0 → 1
  - Возможно небольшое смещение по X (slide-in слева): -20px → 0px
  - Прогресс-бар заполняется слева направо с анимацией ширины

**Исчезновение (fade-out)**:
- Длительность: ~0.33 секунды (10 кадров)
- Easing: ease-in или cubic-bezier(0.42, 0, 1, 1)
- Эффекты:
  - Opacity: 1 → 0
  - Возможно небольшое смещение по X: 0px → -20px

### Анимация появления (приблизительная)

```javascript
// Псевдокод для Remotion или подобной библиотеки
const frame = useCurrentFrame();

// Фаза 1: Появление прогресс-бара (кадры 11-20)
const progressBarOpacity = interpolate(
  frame,
  [11, 20],
  [0, 1],
  { extrapolateRight: 'clamp' }
);

const progressBarWidth = interpolate(
  frame,
  [11, 20],
  [0, 100],
  { extrapolateRight: 'clamp', easing: 'easeOut' }
);

// Фаза 2: Появление текста (кадры 20-30)
const textOpacity = interpolate(
  frame,
  [20, 30],
  [0, 1],
  { extrapolateRight: 'clamp', easing: 'easeOut' }
);

const textTranslateX = interpolate(
  frame,
  [20, 30],
  [-20, 0],
  { extrapolateRight: 'clamp', easing: 'easeOut' }
);

// Фаза 3: Удержание (кадры 31-50)
// Никаких изменений, opacity = 1, translateX = 0

// Фаза 4: Исчезновение (кадры 51-60)
const fadeOutOpacity = interpolate(
  frame,
  [51, 60],
  [1, 0],
  { extrapolateRight: 'clamp', easing: 'easeIn' }
);

const fadeOutTranslateX = interpolate(
  frame,
  [51, 60],
  [0, -20],
  { extrapolateRight: 'clamp', easing: 'easeIn' }
);

// Итоговая прозрачность
const finalOpacity = frame < 51 ? textOpacity : fadeOutOpacity;
const finalTranslateX = frame < 51 ? textTranslateX : fadeOutTranslateX;
```

## Ключевые особенности для плавности

1. **Плавные переходы**: Используйте easing-функции (ease-out для появления, ease-in для исчезновения)
2. **Последовательность**: Сначала появляется прогресс-бар, затем текст
3. **Синхронизация**: Оранжевый блок должности появляется синхронно с именем
4. **Без резких скачков**: Все изменения opacity и position должны быть плавными
5. **Timing**: Появление занимает в два раза больше времени, чем исчезновение

## Рекомендации по реализации в Remotion

```typescript
import { interpolate, useCurrentFrame, spring } from 'remotion';

export const NameCard: React.FC<{ name: string; title: string }> = ({ name, title }) => {
  const frame = useCurrentFrame();
  const fps = 30;

  // Появление (кадры 11-30)
  const appearProgress = spring({
    frame: frame - 11,
    fps,
    config: {
      damping: 100,
      stiffness: 200,
      mass: 0.5,
    },
  });

  // Исчезновение (кадры 51-60)
  const disappearProgress = interpolate(
    frame,
    [51, 60],
    [0, 1],
    { extrapolateRight: 'clamp' }
  );

  // Итоговая прозрачность
  const opacity = frame < 51
    ? appearProgress
    : 1 - disappearProgress;

  // Смещение
  const translateX = frame < 51
    ? interpolate(appearProgress, [0, 1], [-20, 0])
    : interpolate(disappearProgress, [0, 1], [0, -20]);

  return (
    <div
      style={{
        position: 'absolute',
        bottom: 40,
        left: 40,
        opacity,
        transform: `translateX(${translateX}px)`,
      }}
    >
      <div
        style={{
          backgroundColor: '#000',
          color: '#FFF',
          fontWeight: 'bold',
          fontSize: 24,
          padding: '8px 16px',
          fontFamily: 'sans-serif',
          letterSpacing: '1px',
        }}
      >
        {name}
      </div>
      <div
        style={{
          backgroundColor: '#FF8C00',
          color: '#FFF',
          fontWeight: 'bold',
          fontSize: 14,
          padding: '4px 16px',
          fontFamily: 'sans-serif',
          letterSpacing: '0.5px',
        }}
      >
        {title}
      </div>
    </div>
  );
};
```

## Выводы

Анимация представляет собой классический lower-third (нижняя треть) элемент, широко используемый в видеопроизводстве для представления персонажей. Ключевые характеристики:

- **Простота**: Минималистичный дизайн с двумя цветами (черный и оранжевый)
- **Плавность**: Все переходы используют easing-функции для естественного движения
- **Читаемость**: Высокий контраст между текстом и фоном
- **Профессионализм**: Стандартное расположение в левом нижнем углу
- **Оптимальный timing**: Достаточно времени для прочтения информации (около 1 секунды статического отображения)

Данная анимация может быть легко адаптирована для других имен и должностей, а также для различных цветовых схем бренда.
